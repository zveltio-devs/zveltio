Overview

Zveltio's export system allows you to generate documents from your data. Whether you need simple CSV exports or formatted PDF reports, the export system has you covered.

Supported Formats

Format Extension Use Case
CSV .csv Data analysis, spreadsheets
Excel .xlsx Spreadsheets with formatting
PDF .pdf Printable documents, reports
JSON .json Data interchange

API Endpoints

Export Data

POST /api/export
{
  "collection": "products",
  "format": "csv",
  "fields": ["name", "price", "category"],
  "filter": { "category": "electronics" }
}

Parameters

Parameter Type Description
collection string Name of the collection to export
format string Export format: csv, xlsx, pdf, json
fields array Fields to include in export
filter object Filter criteria

Templates

For PDF exports, you can use templates:

POST /api/export
{
  "collection": "invoices",
  "format": "pdf",
  "template": "invoice",
  "data": {
    "invoice_number": "INV-2024-001",
    "date": "2024-01-15"
  }
}
