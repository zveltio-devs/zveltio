Overview

Zveltio provides a unified AI integration layer that supports multiple AI providers. Whether you prefer OpenAI, Anthropic, or self-hosted solutions, Zveltio has you covered.

Conversational AI with memory

📄 Document Intelligence

Extract insights from documents

🌐 AI Translation

Multilingual support

AI Features

Chat

Context-aware conversational AI:

POST /api/ai/chat
{
  "provider": "openai",
  "model": "gpt-4",
  "messages": [
    { "role": "user", "content": "Hello!" }
  ]
}

Embeddings

Generate vector embeddings for semantic search:

POST /api/ai/embed
{
  "provider": "openai",
  "model": "text-embedding-3-small",
  "input": "The quick brown fox"
}

Semantic Search (RAG)

Search documents using natural language:

POST /api/ai/search
{
  "query": "How do I configure authentication?",
  "collection": "documents",
  "limit": 5
}

Configuration

Environment Variables

# OpenAI
OPENAI_API_KEY=sk-...

# Anthropic
ANTHROPIC_API_KEY=sk-ant-...

# Ollama (self-hosted)
OLLAMA_BASE_URL=http://localhost:11434

# Azure OpenAI
AZURE_OPENAI_ENDPOINT=https://your-resource.openai.azure.com/
AZURE_OPENAI_API_KEY=your-api-key

API Endpoints

Endpoint Description
POST /api/ai/chat Chat completion
POST /api/ai/embed Generate embeddings
POST /api/ai/search Semantic search
GET /api/ai/models List available models
