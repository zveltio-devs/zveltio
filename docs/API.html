API Reference

Complete API reference for Zveltio Engine. All endpoints are prefixed with /api and require authentication unless otherwise specified.

Authentication

Zveltio uses Better-Auth for authentication. Sessions are managed via HTTP-only cookies.

POST /api/auth/sign-up/email

Register a new user with email and password.

Request Body

Field Type Description
email string User email address Required
password string User password (min 8 chars) Required
name string User display name Optional 
POST /api/auth/sign-up/email
// Request
{
  "email": "user@example.com",
  "password": "securepassword123",
  "name": "John Doe"
}

// Response
{
  "user": {
    "id": "usr_...",
    "email": "user@example.com",
    "name": "John Doe"
  }
}
POST /api/auth/sign-in/email

Sign in with email and password. Returns session cookie. 

POST /api/auth/sign-in/email
// Request
{
  "email": "user@example.com",
  "password": "securepassword123"
}

// Response - Sets session cookie
{
  "user": { ... }
}
POST /api/auth/sign-out

Sign out and invalidate session.

Collections

Dynamic collections allow you to create database tables at runtime without migrations.

GET /api/collections

List all collections.

GET /api/collections

// Response
{
  "collections": [
    {
      "id": "col_abc123",
      "name": "products",
      "fields": [...],
      "created_at": "2026-03-04T00:00:00Z"
    }
  ]
}
POST /api/collections

Create a new collection.

Request Body

Field Type Description
name string Collection name (snake_case) Required
fields array Array of field definitions Required 
POST /api/collections
{
  "name": "products",
  "fields": [
    { "name": "title", "type": "text", "required": true },
    { "name": "price", "type": "number", "required": true },
    { "name": "description", "type": "richtext" }
  ]
}

Data CRUD

Perform CRUD operations on collection records.

GET /api/data/:collection

List records from a collection with filtering and pagination.

Query Parameters

Parameter Type Description
limit number Number of records (default: 50) Optional
offset number Pagination offset Optional
filter string JSON filter expression Optional
sort string Sort field (e.g., "created_at:desc") Optional 
GET /api/data/products?limit=10&filter={"category":"electronics"}

// Response
{
  "records": [...],
  "total": 100,
  "limit": 10,
  "offset": 0
}
POST /api/data/:collection

Create a new record in a collection. 

POST /api/data/products
{
  "title": "iPhone 15 Pro",
  "price": 999,
  "category": "electronics"
}

// Response
{
  "record": {
    "id": "rec_xyz789",
    "title": "iPhone 15 Pro",
    "price": 999,
    "created_at": "2026-03-04T12:00:00Z"
  }
}
GET /api/data/:collection/:id

Get a single record by ID.

PUT /api/data/:collection/:id

Update an existing record.

DELETE /api/data/:collection/:id

Delete a record (soft delete by default).

Permissions

Manage user roles and permissions using Casbin RBAC.

POST /api/permissions/assign-role

Assign a role to a user.

POST /api/permissions/assign-role
{
  "userId": "usr_abc123",
  "role": "admin"
}

// Available roles: god, admin, manager, employee, guest

Storage

File upload and management via S3-compatible storage.

POST /api/storage/presign

Get a presigned URL for direct upload to S3. 

POST /api/storage/presign
{
  "filename": "document.pdf",
  "contentType": "application/pdf"
}

// Response
{
  "uploadUrl": "https://...",
  "fileId": "file_xyz",
  "publicUrl": "https://..."
}

AI Chat

Conversational AI with context awareness.

POST /api/ai/chat

Send a chat message to the AI assistant. 

POST /api/ai/chat
{
  "message": "What are the top selling products?",
  "context": { "collection": "products" }
}

// Response
{
  "response": "Based on your sales data, the top selling products are...",
  "sources": ["products table"]
}

Semantic Search

RAG-powered document search using vector embeddings.

POST /api/ai/search

Perform semantic search across collections. 

POST /api/ai/search
{
  "query": "How to reset password",
  "collection": "articles",
  "limit": 5
}

// Response
{
  "results": [
    {
      "id": "rec_123",
      "title": "Password Reset Guide",
      "content": "To reset your password...",
      "score": 0.95
    }
  ]
}

Embeddings

Generate vector embeddings for AI-powered features.

POST /api/ai/embeddings/generate

Generate a single embedding.

POST /api/ai/embeddings/generate
{
  "text": "The quick brown fox jumps over the lazy dog"
}

// Response
{
  "embedding": [0.123, -0.456, 0.789, ...]
}
POST /api/ai/embeddings/batch

Generate embeddings for multiple texts.

Webhooks

Configure event-driven integrations.

GET /api/webhooks

List all configured webhooks.

POST /api/webhooks

Create a new webhook.

POST /api/webhooks
{
  "name": "Slack Notifications",
  "url": "https://hooks.slack.com/services/...",
  "events": ["record.created", "record.updated"],
  "active": true
}

Export

Generate reports in multiple formats.

POST /api/export/:collection

Export collection data to PDF, Excel, or CSV. 

POST /api/export/products
{
  "format": "pdf",
  "filters": { "category": "electronics" },
  "template": "professional"
}

// Response - Returns file download

Translations

Internationalization and translation management.

GET /api/translations/:key

Get translation for a key in all languages.

POST /api/translations

Create or update a translation.
