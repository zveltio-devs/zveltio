Installation Issues

Docker services won't start

Error: Container fails to start or exits immediately 
# Check container logs
docker compose logs engine

# Check if ports are already in use
netstat -ano | findstr "3000"

Bun not found

Error: bun: command not found 
# Install Bun
curl -fsSL https://bun.sh/install | bash

# Verify installation
bun --version

Database Issues

Database connection refused

Error: connect ECONNREFUSED 127.0.0.1:5432 
# Check if PostgreSQL is running
docker compose ps db

# Start the database
docker compose up -d db

# Check database logs
docker compose logs db

Connection pool exhausted

Error: Remaining connection slots are reserved 
# Check active connections
docker compose exec db psql -U zeltio -d zveltio -c "SELECT count(*) FROM pg_stat_activity"

Authentication Issues

Session expired

Error: Session token is invalid or expired 
# Check BETTER_AUTH_SECRET in .env
# Make sure it's at least 32 characters

# Clear browser cookies and try again

Cannot create God user

Error: Failed to create God user 
# Make sure database is running
docker compose up -d db pooler

# Run migrations first
bun run -T packages/engine/src/db/migrate.ts

# Then create God user
bun run packages/cli/src/index.ts create-god

Permission denied errors

Error: Forbidden - insufficient permissions 
# Check user role in database
docker compose exec db psql -U zeltio -d zveltio -c "SELECT id, email, role FROM \"user\""

# Update user role to god if needed
docker compose exec db psql -U zeltio -d zveltio -c "UPDATE \"user\" SET role = 'god' WHERE email = 'admin@example.com'"

Performance Issues

Slow API responses

Symptom: API requests take more than 1 second
  1. Check database query performance
  2. Monitor Valkey cache hit rate
  3. Review Prometheus metrics at /metrics
  4. Check disk I/O

High memory usage

Symptom: Engine uses more than 2GB RAM 
# Check memory usage
docker stats

# Restart engine to clear memory
docker compose restart engine

Deployment Issues

SSL certificate errors

Error: SSL certificate verification failed 
# Check certificate expiration
openssl s_client -connect yourdomain.com:443 -showcerts

# Renew Let's Encrypt certificate
docker compose restart nginx

Webhook delivery failures

Symptom: Webhooks not reaching destination
  1. Check webhook logs in Grafana dashboard
  2. Verify destination server is reachable
  3. Check webhook secret is correct
  4. Review retry settings

Getting Help

If you're still experiencing issues:

  1. Check the GitHub Issues
  2. Search existing discussions
  3. Open a new issue with detailed logs
