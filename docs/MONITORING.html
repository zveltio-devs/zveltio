Prometheus

Metrics are available at the /metrics endpoint:

# Access metrics
curl http://localhost:3000/metrics

Prometheus configuration:

# prometheus.yml
scrape_configs:
  - job_name: 'zveltio'
    static_configs:
      - targets: ['engine:3000']

Grafana

Access Grafana at http://localhost:3001

Default credentials: admin/admin

Dashboards

Zveltio includes pre-configured dashboards:

Dashboard Description
Zeltio Overview System-wide metrics and health
AI Usage AI provider usage and costs
Webhooks Webhook delivery status and timing

Available Metrics

Metric Type Description
zveltio_http_requests_total Counter Total HTTP requests
zveltio_http_request_duration_seconds Histogram Request duration
zveltio_db_query_duration_seconds Histogram Database query duration
zveltio_cache_hits_total Counter Cache hit count
zveltio_cache_misses_total Counter Cache miss count
